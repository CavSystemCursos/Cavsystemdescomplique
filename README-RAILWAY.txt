DEPLOY NA RAILWAY

1. Extraia o ZIP
2. Envie para um repositório GitHub
3. Entre em https://railway.app
4. Clique em "Deploy from GitHub repo"
5. Selecione o projeto
6. Railway irá detectar automaticamente

Se pedir:
Build Command:
pnpm build

Start Command:
pnpm start
